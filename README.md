# Lehema - Sitio web

Sitio web estático para GitHub Pages de Lehema, marca dedicada al diseño, confección y venta de pijamas, además de servicios de costura y arreglos.

## Archivos principales

- `index.html`: estructura principal del sitio.
- `styles.css`: estilos visuales y responsive.
- `robots.txt`: reglas para buscadores.
- `sitemap.xml`: mapa del sitio para SEO.
- `CNAME`: dominio personalizado para GitHub Pages.
- `assets/logo.svg`: logo base editable.

## Ajustes recomendados

Antes de publicar, actualiza en `index.html`:

- Número de WhatsApp.
- Correo de contacto.
- Enlace de Instagram.
- Textos de dirección o ciudad, si aplica.
- Fotografías reales de productos, si deseas reemplazar los bloques visuales.

## Publicación

Sube estos archivos al repositorio de GitHub Pages, igual que el sitio anterior. En la configuración del repositorio, valida que GitHub Pages esté activo y que el dominio personalizado sea `lehema.com`.
